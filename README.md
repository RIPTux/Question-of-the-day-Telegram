## Question of the day Telegram
Это телеграмм бот отправляющий в группы "Вопрос дня" каждые 24. 
Бот работает по вайт листу и отсчет для отправки идет от отправки команды /question в группе!

# Технологии:
  1) Бот работает на основе SQLite
  2) Используется автосоздавемый кофиг YAML
  3) Бот использует 21 java
# Команды:
  - Команды:
      - /question || +опрос - бот отправляет в чат "вопрос дня"
        
  - Админ команды:
     - Админ команды может выполнить только создатель бота айди которого указывается в конфиге
     - /configReload - перезагрузка бота
     - /add <ChatId> - добавление группы в вайт лист
     - /delGroup <ID> - деактивирует группу в вайт листе
     - /getGroup - показывает все группы находящиеся в вайт листе и их состояние
     - /getTime - показывает список групп и время последней отправки "вопроса дня"
     - /getQuestion - показывает все вопросы находящиеся в базе
   
# Инструкция по установке:
  1) Скачиваем архив с ботом
  2) Перекидываем все файлы с архива в нужную директорию
  3) Запускаем бота командой java -jar questionOfTheDayTelegram-1.1.jar
  4) При первом запуске бот выключится после создания конфига
  5) Заполняем конфиг
  6) Снова запускаем бота
  7) Поздравляю вы запустили бота!
